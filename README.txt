==================================
EEA ElasticSearch RDF River Plugin
==================================

Introduction
============

The EEA RDF River Plugin for ElasticSearch allows to harvest metadata from
SPARQL endpoints or plain RDF files into ElasticSearch. It is provided as a
plugin.
